# simple-os-web

🖥️ 浏览器里的 macOS 模拟器：桌面、Dock、访达、终端，还能玩游戏和上网。

## ✨ 功能
- 仿 macOS 桌面、Dock 栏、菜单栏  
- 窗口可拖拽、缩放、最小化、最大化  
- 🌙 深色模式切换  
- 🌐 Safari 浏览器可**真实访问网页**（通过 iframe）  
- 🎮 内置小游戏：贪吃蛇、2048、扫雷  
- 🖼️ 可切换壁纸（默认/海洋/极光）  
- ⌨️ 快捷键支持：`⌘/Ctrl + K` 打开终端，`⌘/Ctrl + N` 打开访达，`⌘/Ctrl + Space` 搜索  

## 🌐 在线体验
👉 **[点击立即体验](https://LuckJane.github.io/simple-os-web/)**

## 🚀 本地运行
直接下载 `index.html`，用浏览器打开即可，无需安装任何依赖。
